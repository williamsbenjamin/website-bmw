NFLSimulatoR is an R package for simulating plays and drives using play-by-play data from the National Football League. The simulations are generated by sampling play-by-play data from previous football seasons, adding statistical rigor to any decisions or inferences arising from the simulations. The package is particularly useful as a data-driven tool for evaluating potential in-game strategies or rule changes within the league.
